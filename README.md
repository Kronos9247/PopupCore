# PopupCore V2

Popup Core is a simple extension of expression2. With popup core you can create Derma and Vguis in e2. 

<h2>UI Types:</h2>
<ul>
  <li>Frame</li>
  <li>Button</li>
  <li>ComboBox</li>
  <li>Html Box</li>
  <li>Label</li>
</ul>
